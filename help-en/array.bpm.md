# array.bpm

###### BPM calculator

*available since version:* 0.4

---


## information
Calculates BPM (Beats Per Minute) rate using SoundTouch library


[![example](../examples/img/array.bpm.jpg)](../examples/pd/array.bpm.pd)

_type:_ symbol<br>

Get/set array name<br>
_type:_ symbol<br>
