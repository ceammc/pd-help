simple array player for using with vline~

---

---
arguments:

NAME: array name<br>

---
properties:

@array: array name<br>
@state: 
            player state<br>
@speed: play speed<br>
@begin(samp): begin position in
            samples<br>
@end(samp): end position in
            samples<br>
@abs_begin(samp): 
            absolute start position in samples<br>
@abs_end(samp): 
            absolute stop position in samples<br>
@reversed: playing backwards (if set
            to 1)<br>
