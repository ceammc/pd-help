# conv.list2props


converts list to series of property messages

```


[list 1 2 3 @prop1 100 @prop2 A B C @prop3(
|
[list->props {w=20}      ]
|               |       ^|
[route @prop1]  [print] [ui.display]
|
[F]

            
```
