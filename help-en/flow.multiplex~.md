# flow.multiplex~

---

## arguments:

* **N**
number of signals inputs<br>
__type:__ int<br>

__type:__ list<br>
__default:__ 1 0<br>



## inlets:

* first input signal 
__type:__ audio<br>
* last input signal 
__type:__ audio<br>
* multiplexer control

## outlets:

* output
__type:__ audio<br>
