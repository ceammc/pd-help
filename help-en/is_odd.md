# is_odd

###### checks if input number is odd

*available since version:* 0.1

---




[![example](../examples/img/is_odd.jpg)](../examples/pd/is_odd.pd)









## inlets:

* outputs &#39;1&#39; if number is not a multiple of two, otherwise outputs
                &#39;0&#39;. 
__type:__ control<br>



## outlets:

* 0 or 1
__type:__ control<br>
* passed odd numbers
__type:__ control<br>
