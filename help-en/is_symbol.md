# is_symbol

###### checks if input data has *symbol* type

*available since version:* 0.1

---

* outputs &#39;1&#39; to 1st outlet, otherwise &#39;0&#39;<br>
_type:_ control



## outlets:

* 0 or 1<br>
_type:_ control
* passes *symbol* message<br>
_type:_ control
