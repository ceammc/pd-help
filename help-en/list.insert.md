# list.insert


insert atom or list to the specified position of input list

---

Works with data atoms<br>


---


```


[1 2 3 4(
|
|  [list D E F(  [F]
|  |             |
|  |.            |..
[list.insert A B C @index 1]
|
[ui.display @display_type=1]

            
```
