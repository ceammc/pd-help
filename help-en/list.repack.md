# list.repack

###### repack list to specified size

*available since version:* 0.1

---




[![example](../examples/img/list.repack.jpg)](../examples/pd/list.repack.pd)



## arguments:

* **SIZE**
output list size<br>
_type:_ int<br>





## properties:

* **@size** 
Get/set output list size<br>
_type:_ int<br>
_range:_ 1..1024<br>
_default:_ 1<br>



## inlets:

* input list<br>
_type:_ control



## outlets:

* output list<br>
_type:_ control
* bang after last element output<br>
_type:_ control
