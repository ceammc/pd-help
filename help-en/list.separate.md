# list.separate


sends list elements one by one separately

---

<br>


---


```


[1 2 3 4 5( [\(a b c d e\), bang(
|           |
|           [ml] [_]
|           |    |
|           |    [@enumerate $1(
|           |    |
[list.separate   ]
|
[print SEPARATE]

            
```

---
arguments:


---
properties:

@enumerate: enumeration mode. If
            enumeration is active output pairs: INDEX ATOM, otherwise single atoms<br>
@from: enumeration starting index<br>
