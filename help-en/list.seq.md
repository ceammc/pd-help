# list.seq


numeric sequence list generator

```


[B]            [1 -1 0.1(
|       [2 10( |
|  [F]  |      |
|  |    |      |
[list.seq 3 10 2]
|
[m set]
|
[ (

            
```

---
arguments:

FROM: start value<br>
TO: end value<br>
STEP: sequence step<br>

---
properties:

@from: start value<br>
@to: end value<br>
@step: sequence step (always
            positive)<br>
@closed: generate closed range sequence<br>
