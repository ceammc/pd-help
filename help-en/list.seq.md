# list.seq

###### numeric sequence list generator

*available since version:* 0.1

---




[![example](../examples/img/list.seq.jpg)](../examples/pd/list.seq.pd)



## arguments:

* **FROM**
start value<br>
__type:__ float<br>

* **TO**
end value<br>
__type:__ float<br>

* **STEP**
sequence step<br>
__type:__ float<br>





## properties:

* **@from** 
Get/set start value<br>
__type:__ float<br>
__default:__ 0<br>

* **@to** 
Get/set end value<br>
__type:__ float<br>
__default:__ 1<br>

* **@step** 
Get/set sequence step (always positive)<br>
__type:__ float<br>
__default:__ 1<br>

* **@closed** 
Get/set generate closed range sequence<br>
__type:__ flag<br>
__default:__ 0<br>



## inlets:

* outputs generated list 
__type:__ control<br>



## outlets:

* generated sequence
__type:__ control<br>
