# local.list


local named list object

---

Defines local scope list variable, accessed by ID name and available only within
            its patch (canvas).
<br>


---

---
arguments:

ID: object ID<br>

---
properties:

@empty: 1 if list is
            empty, otherwise 0<br>
@size: number of
            elements in list<br>
