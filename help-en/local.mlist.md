# local.mlist


local named mlist object

---

Defines local scope mlist variable, accessed by ID name and available only within
            its patch (canvas).
<br>


---


![example](examples/local.mlist-example.jpg)

---
arguments:

ID: object ID<br>

---
properties:

@empty: 1 if list is
            empty, otherwise 0<br>
@size: number of
            elements in list<br>
