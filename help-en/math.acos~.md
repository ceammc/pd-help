# math.acos~

###### arc cosine function for signals

*available since version:* 0.9

---


## information
Outputs the principle value of the arc cosine of input signal. The result is in
            the range [0, pi]



[![example](../examples/img/math.acos~.jpg)](../examples/pd/math.acos~.pd)









## inlets:

* input signal 
__type:__ audio<br>

* result signal
