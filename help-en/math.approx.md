# math.approx


check if input value approximately equal to pattern

---

checks is |x - pattern| &lt;= |epsilon|
<br>


---


```


[HS]
|
|   [1.99 2 0.01(
|   |
[F] |
|   |
[math.approx 50 10] /* 40 <= x <= 60 */
|
[F]

            
```

---
arguments:

P: compare pattern<br>
E: compare epsilon<br>

---
properties:

@value: compare pattern<br>
@epsilon: compare epsilon<br>
