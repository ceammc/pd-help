# math.asin~

###### arc sine for signals

*available since version:* 0.9

---


## information
Outputs the principle value of the arc sine of input signal. The result is in the
            range [-pi/2, +pi/2]



[![example](../examples/img/math.asin~.jpg)](../examples/pd/math.asin~.pd)









## inlets:

* input signal 
__type:__ audio<br>



## outlets:

* result signal
__type:__ audio<br>
