# math.atan

###### arc tangent function

*available since version:* 0.1

---


## information
Outputs the principal value of the arc tangent of input value. The result is in
            the range [-π/2, +π/2].
Special values:
atan(+-0) returns +-0.
atan(+-infinity) returns +-π/2
