# math.sync_and


sync logical AND operation

```


[_]   [_]   /*same as:*/ [_]  [_]      /*0 0 | 0*/
|     |                  |    |
|     |                  |    [t b f]  /*0 1 | 0*/
|     |.                 |    | ^|.
[&&'    ]                [&&      ]    /*1 0 | 0*/
|                        |
[_]                      [_]           /*1 1 | 1*/

            
```
