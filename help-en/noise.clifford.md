# noise.clifford


Part of a-chaos library

---

Clifford Attractors

Attributed to Cliff Pickover
---
 
See also Peter de Jong attractors 
 
 xn+1 = sin(a yn) + c cos(a xn)
 
 yn+1 = sin(b xn) + d cos(b yn)
 
 where a, b, c, d are variabies that define each attractor.
