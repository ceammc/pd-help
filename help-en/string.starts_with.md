# string.starts_with

*available since version:* 0.3

---

* **PREFIX**

_type:_ symbol<br>





## properties:

Get/set prefix<br>
_type:_ symbol<br>



## inlets:

* input string or symbol<br>
_type:_ control
* search prefix (string or symbol)<br>
_type:_ control



## outlets:

* output 1 if input string starts with prefix, otherwise 0<br>
_type:_ control
