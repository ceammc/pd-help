# array.bpm

###### BPM calculator

*доступно с версии:* 0.4

---


## информация
Calculates BPM (Beats Per Minute) rate using SoundTouch library


[![example](../examples/img/array.bpm.jpg)](../examples/pd/array.bpm.pd)



## аргументы:

* **ARRAY**
array name<br>
_тип:_ symbol<br>





## свойства:

* **@array** 
Запросить/установить array name<br>
_тип:_ symbol<br>



## входы:

* starts calculation<br>
_тип:_ control



## выходы:

* float value - BPM<br>
_тип:_ control
