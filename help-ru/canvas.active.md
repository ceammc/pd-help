# canvas.active

###### checks if canvas window is on top and active

*доступно с версии:* 0.9.6

---




[![example](../examples/img/canvas.active.jpg)](../examples/pd/canvas.active.pd)

_тип:_ bool<br>
_по умолчанию:_ 0<br>



## входы:

* output canvas active state<br>
_тип:_ control
