# dict.get

###### output dict values

*доступно с версии:* 0.6

---




[![example](../examples/img/dict.get.jpg)](../examples/pd/dict.get.pd)



## аргументы:

* **KEYS**
keys<br>
_тип:_ list<br>





## свойства:

* **@keys** (initonly)
Запросить/установить extracted keys<br>
_тип:_ list<br>

* **@default** 
Запросить/установить default value when key is not found in dictionary. If not specified no output
is performed.<br>
_тип:_ atom<br>
_по умолчанию:_ null<br>



## входы:

* input dictionary<br>
_тип:_ control



## выходы:

* first key output<br>
_тип:_ control
* ... key output<br>
_тип:_ control
* nth key output<br>
_тип:_ control
