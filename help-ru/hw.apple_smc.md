# hw.apple_smc

###### query the system management control in Apples

*доступно с версии:* 0.8

---




[![example](../examples/img/hw.apple_smc.jpg)](../examples/pd/hw.apple_smc.pd)





## методы:

* **read**
query specified SMC key<br>
  __параметры:__
  - **KEY** SMC key<br>
    тип: symbol <br>
    обязательно: True <br>
