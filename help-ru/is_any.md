# is_any

###### checks if input data has *any* type

*доступно с версии:* 0.1

---




[![example](../examples/img/is_any.jpg)](../examples/pd/is_any.pd)









## входы:

* outputs &#39;1&#39; to 1st outlet, otherwise &#39;0&#39;<br>
_тип:_ control



## выходы:

* 0 or 1<br>
_тип:_ control
* passes *any* message<br>
_тип:_ control
