# is_bang

###### checks if input data has *bang* type

*доступно с версии:* 0.1

---




[![example](../examples/img/is_bang.jpg)](../examples/pd/is_bang.pd)









## входы:

* outputs &#39;1&#39; to 1st outlet, otherwise &#39;0&#39;<br>
_тип:_ control



## выходы:

* 0 or 1<br>
_тип:_ control
* passes *bang* message<br>
_тип:_ control
