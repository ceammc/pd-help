# is_symbol

###### checks if input data has *symbol* type

*доступно с версии:* 0.1

---




[![example](../examples/img/is_symbol.jpg)](../examples/pd/is_symbol.pd)









## входы:

* outputs &#39;1&#39; to 1st outlet, otherwise &#39;0&#39;<br>
_тип:_ control



## выходы:

* 0 or 1<br>
_тип:_ control
* passes *symbol* message<br>
_тип:_ control
