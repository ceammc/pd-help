# list.runsum

###### running sum

*доступно с версии:* 0.9.4

---


## информация
Computes the partial sums of the elements in the subranges of the input list • y0 = x0 • y1 = x0 + x1 • y2 = x0 + x1 + x2 • y3 = x0 + x1 + x2 + x3 etc...
