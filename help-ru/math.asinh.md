# math.asinh

###### inverse hyperbolic sine function

*доступно с версии:* 0.1

---


## информация
The asinh() function computes the inverse hyperbolic sine of the real argument Special values: asinh(+-0) returns +-0. asinh(+-infinity) returns +-infinity.
