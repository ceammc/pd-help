# math.asin~

###### arc sine for signals

*доступно с версии:* 0.9

---


## информация
Outputs the principle value of the arc sine of input signal. The result is in the range [-pi/2, +pi/2]


[![example](../examples/img/math.asin~.jpg)](../examples/pd/math.asin~.pd)









## входы:

* input signal<br>
_тип:_ audio



## выходы:

* result signal<br>
_тип:_ audio
