# math.atan

###### arc tangent function

*доступно с версии:* 0.1

---


## информация
Outputs the principal value of the arc tangent of input value. The result is in the range [-π/2, +π/2]. Special values: atan(+-0) returns +-0. atan(+-infinity) returns +-π/2
