# math.atan~

###### arc tangent for signals

*доступно с версии:* 0.9

---


## информация
Outputs arc tangent of input signal. The result is in the range [-π/2, +π/2]


[![example](../examples/img/math.atan~.jpg)](../examples/pd/math.atan~.pd)









## входы:

* input signal<br>
_тип:_ audio



## выходы:

* result signal<br>
_тип:_ audio
