# math.sqrt

###### square root function

*доступно с версии:* 0.1

---


## информация
The sqrt() function compute the non-negative square root of x. Special values: sqrt(-0) returns -0. sqrt(x) returns a NaN and generates a domain error for x &lt; 0.
