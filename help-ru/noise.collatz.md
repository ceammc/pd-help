## методы:

* **offset**
<br>
  __параметры:__
  - **OFFSET** offset value<br>
    тип: float <br>
    обязательно: True <br>

* **mode**
<br>
  __параметры:__
  - **MODE** mode value. 0 for Collatz, 1 for Terras<br>
    тип: int <br>
    обязательно: True <br>

* **om**
<br>
  __параметры:__
  - **OM** enables output when cut or fold value is changed<br>
    тип: int <br>
    обязательно: True <br>
