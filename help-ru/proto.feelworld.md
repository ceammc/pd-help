# proto.feelworld

###### protocol for FeelWorld LivePro L1 HDMI mixer

*доступно с версии:* 0.9.5

---




[![example](../examples/img/proto.feelworld.jpg)](../examples/pd/proto.feelworld.pd)





## методы:

* **input**
select HDMI input<br>
  __параметры:__
  - **N** input index<br>
    тип: int <br>
    обязательно: True <br>






## входы:

* commands<br>
_тип:_ control



## выходы:

* list: bytes to send to UDP port 1000<br>
_тип:_ control
