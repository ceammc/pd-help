# route.prop

###### separate property messages from other types

*доступно с версии:* 0.9.6

---




[![example](../examples/img/route.prop.jpg)](../examples/pd/route.prop.pd)









## входы:

* property to *first* outlet, any other messages to second outlet<br>
_тип:_ control



## выходы:

* property output<br>
_тип:_ control
* other messages<br>
_тип:_ control
