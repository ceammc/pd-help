## аргументы:

* **VALUE**
initial string<br>
_тип:_ symbol<br>



## методы:

* **clear**
clear string without output<br>

* **set**
set string content without output<br>

* **append**
append values to the end of string<br>




## свойства:

* **@value** 
Запросить/установить string value<br>
_тип:_ list<br>



## входы:

* outputs string value<br>
_тип:_ control



## выходы:

_тип:_ control
