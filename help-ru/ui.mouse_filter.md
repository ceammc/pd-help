# ui.mouse_filter

###### filter mouse events from ui objects

*доступно с версии:* 0.8

---


## информация
pass only specified mouse events from ui.* objects


[![example](../examples/img/ui.mouse_filter.jpg)](../examples/pd/ui.mouse_filter.pd)
